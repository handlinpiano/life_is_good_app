# Task: Add More Cities to the Dropdown

## Difficulty: Easy | Time: 10-15 minutes

## File to Edit
`frontend/src/utils/constants.js`

## Your Task
Add **50+ more cities** to the `MAJOR_CITIES` array.

### India (add at least 30)
- All state capitals
- Pilgrimage cities (Haridwar, Rishikesh, Tirupati, Shirdi, Ujjain, Puri)
- Historical cities (Agra, Lucknow, Mysore)

### International (add at least 20)
- USA, UK, Canada, Australia, UAE, Singapore, etc.

## Format
```javascript
{ name: 'City, Country', lat: XX.XXXX, lng: XX.XXXX }
```

## Requirements
- Alphabetical order within regions
- Accurate coordinates (4 decimal places)
- No duplicates
